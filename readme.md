### 🌟 English Section

This project, AWS Bedrock AI Agent, exemplifies advanced cloud-native integration for WhatsApp Business AI integration. Its modular design supports broad scalability aligned with businesses' specific service pipelines. Key focus areas include streamlined automated workflows, real-time language processing, and secure, dependable infrastructure.

## 🇺🇸 Project Overview

The AWS Bedrock project is designed to create an AI-driven Whatsapp Business Agent utilizing several cloud technologies. This initiative follows a modular architecture to allow scalability across diverse business niches.

### 🏗 Architecture:
1. **Infrastructure as Code:** Managed entirely through **Terraform** for repeatability and version control.
2. **API Gateway:** Implemented as the entry point for seamless communication.
3. **Lambda Functions:** Automated responses managed by serverless functions.
4. **Data Storage:** Combines the use of **S3** for unstructured data and **Aurora PostgreSQL** for relational data.

## 🇧🇷 Visão Geral do Projeto

O projeto AWS Bedrock foi desenvolvido para criar um agente do WhatsApp Business com AI, utilizando tecnologias modernas de nuvem. A arquitetura modular permite escalabilidade em diferentes nichos comerciais.

### 🏗 Arquitetura:
1. **Infraestrutura como Código:** Gerenciado completamente com **Terraform** para controle de versão e repetibilidade.
2. **API Gateway:** Funciona como ponto de entrada seguro para comunicação.
3. **Funções Lambda:** Automatização de respostas utilizando funções serverless.
4. **Armazenamento de Dados:** Combina o uso do **S3** para dados não-estruturados e **Aurora PostgreSQL** para dados relacionais.
